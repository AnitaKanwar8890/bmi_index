# BMI Calculator

A simple web-based BMI (Body Mass Index) Calculator that allows users to input their height and weight to determine their BMI category.

## Features
- User-friendly interface.
- Calculates BMI based on height (in cm) and weight (in kg).
- Provides a BMI category: Underweight, Normal, or Overweight.
- Responsive design for a smooth user experience.

## Installation & Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/bmi-calculator.git
2. Navigate to the project directory:
cd bmi-calculator

3. Open index.html in a web browser.

 Files Overview
index.html - Main structure of the BMI calculator.
style.css - Styles for enhancing the user interface.

How It Works
Enter your height in cm.
Enter your weight in kg.
Click the "Calculate" button.
The result will display your BMI and weight category.
